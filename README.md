# 🎓 Smart Student Performance Predictor

An AI-powered Student Performance Prediction System built using **Python, Streamlit, Scikit-learn, and SQLite**. The application predicts student academic performance using Machine Learning, tracks historical performance, provides personalized improvement suggestions, and generates downloadable reports.

---

## ✨ Features

- 🤖 AI-powered student performance prediction
- 📊 Academic performance analytics
- 📈 Performance improvement tracking
- 📝 Personalized study recommendations
- 📄 Downloadable performance reports
- 🗄 SQLite database integration
- 📚 Student history management
- 🎯 Target score planning

---

## 🛠 Technologies Used

- Python
- Streamlit
- Scikit-learn
- Pandas
- NumPy
- Matplotlib
- SQLite
- Joblib

## 🚀 Installation

Clone the repository:

```bash
git clone https://github.com/YOUR_USERNAME/Student-Performance-Predictor.git
```

Move into the project:

```bash
cd Student-Performance-Predictor
```

Install dependencies:

```bash
pip install -r requirements.txt
```

Run the application:

```bash
streamlit run app.py
```
